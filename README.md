# Typing Rush
This is a simple static website which i made by using HTML , CSS and JS . This is a simple typing speed tester website . Where we can measure our WPM , CPM also how many mistakes you done while
typing. It also has sound effects according to your correct typing and wrong typing .

## Technologies Used:
- HTML: HTML is used in this project to insert HTML elements and also make the webpage layout.
- CSS: CSS is used in this project  to apply styling for the html elements and make the webpage look interactive .
- JS: JS is used in this project to impliment the logic and track the WPM , Mistakes and CMP .
